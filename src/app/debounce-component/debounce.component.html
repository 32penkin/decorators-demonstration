Debounce decorator

Debounce a function when you want it to execute only once after a defined interval of time. If the event occurs multiple times within the interval, the interval is reset each time.

To demonstrate the work of the decorator, 2 buttons have been added to increase the counter by 1. One of them increases linearly, the second one is delayed and run function after delay.
